# assignment_F1
prerequisite
* install java 1.8.x
* install gradle

How to execute
* git clone
* cd assignment_F1
* gradle clean run -q
* please input number of car and lengt of race track with Integer value
* result will be printed out !!

How to run unit-test
* cd assignment_F1
* gradle clean test jacoco
* cd build/reports/jacoco/html
* open index.html
